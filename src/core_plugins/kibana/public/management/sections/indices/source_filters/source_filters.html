源过滤器

源过滤器用于在提取日志时排除一个或多个字段。 当在数据分析功能中查看记录时，或是在仪表盘中显示搜寻结果的表格时，适用于此功能。 每行使用单个记录的源代码构建，如果你的日志中有大量的或不重要的字段，你可以从这个较低级别筛选过滤出来。

Note that multi-fields will incorrectly appear as matches in the table below. These filters only actually apply to fields in the original source document, so matching multi-fields are not actually being filtered. 请注意，输入多个字段不会过滤出正确的匹配项。 这些过滤器仅实际应用于源日志中的字段，因此匹配多字段实际上不会被过滤。
